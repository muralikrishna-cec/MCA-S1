JavaScript String and Math Functions

String Methods

String Methods are predefined functions that allow you to manipulate or analyze strings.

1. length

The length property returns the number of characters in a string.

let str = "Hello";
console.log(str.length);
Output: 5

2. toUpperCase()

The toUpperCase() method converts a string to uppercase letters.

let str = "hello";
console.log(str.toUpperCase());
Output: HELLO

3. toLowerCase()

The toLowerCase() method converts a string to lowercase letters.

let str = "HELLO";
console.log(str.toLowerCase());
Output: hello

4. indexOf()

The indexOf() method returns the index of the first occurrence of a specified value in a string.

let str = "Hello, world!";
console.log(str.indexOf("world"));
Output: 7

5. substring()

The substring() method extracts a portion of a string between two specified indices.

let str = "Hello, world!";
console.log(str.substring(7, 12));
Output: world

Math Methods

Math Methods are predefined functions that allow you to perform mathematical operations.

1. Math.max()

The Math.max() method returns the largest of the numbers given as input.

console.log(Math.max(5, 10, 3));
Output: 10

2. Math.min()

The Math.min() method returns the smallest of the numbers given as input.

console.log(Math.min(5, 10, 3));
Output: 3

3. Math.random()

The Math.random() method generates a random number between 0 (inclusive) and 1 (exclusive).

console.log(Math.random());
Output: A random number like 0.3745426870378369

4. Math.floor()

The Math.floor() method rounds a number downward to the nearest integer.

console.log(Math.floor(3.7));
Output: 3

5. Math.ceil()

The Math.ceil() method rounds a number upward to the nearest integer.

console.log(Math.ceil(3.7));
Output: 4

6. Math.pow()

The Math.pow() method returns the base to the exponent power (base^exponent).

console.log(Math.pow(2, 3));
Output: 8
